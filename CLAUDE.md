# Hard Rules
- Classify intent before acting: "I will [action] because [reason]."
- No uncommanded commits. No type suppression. No speculation about unread code.
- No evidence = not complete. Run it, test it, verify it.
- After 3 failures: stop, revert, consult oracle, ask user.
- For 2+ file tasks, prefer delegation to specialist agents.
